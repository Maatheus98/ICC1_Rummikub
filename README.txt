Este trabalho é uma implementação do jogo “Rummikub”
através da linguagem C. O jogo consiste num conjunto
de 106 peças, sendo 104 delas numeradas e 2 coringas.
Ganha o jogador que conseguir descartar todas as 
cartas. Caso acabe as cartas do baralho, será feita a
contagem de pontos onde quem tiver menos pontos ganha.
